JavaScript time

MMP 310 > Week 8

There are cases in programming where you want something to take a certain amount of time to happen, or wait for a certain amount of time to execute.

In JavaScript, there are a few functions that can help us keep track of time.

We can get the time using the object Date(); and it's methods:

Date();
Date.now();
var today = new Date(Date());
today.getDate();
today.getDay();
today.getFullYear();
today.getHours();
today.getMillliseconds();
today.getMonth();
today.getSeconds();
today.getTime();
var yesterday = new Date("Sun Mar 27 2016");

JavaScript also has timers, including setTimeout and setInterval.

setTimeout delays the execution of a function for a specified amount of time measured in milliseconds.

var delayedGreeting = function() {
	alert("Hello world!");
};
setTimeout(delayedGreeting, 1000);

We can save a timeout as a variables and use that variable to cancel it.

var delayedGreeting = function() {
	alert("Hello world!");
};
var g = setTimeout(delayedGreeting, 1000);
clearTimeout(g);

setInterval is used to execute a function repeatedly at a specific interval.

We can create a count down as an example:

var count = 100;
var countdown = function() {
	if (count == 0) clearInterval(countdownInterval);
	console.log(count);
	count--;
};
var countdownInterval = setInterval(countdown, 100);
