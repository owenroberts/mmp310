jQuery is a JavaScript library that simplifies scripting for HTML pages using DOM element selectors and a set of animations, transitions and other manipulations based on CSS and HTML. jQuery was released in 2006 and has become popular in front end web development. It is largely consistent and compatible with all the major browsers.

You can download jQuery from their site and attach the file to your HTML pages with a script tag.

Google also hosts versions of jQuery and other libraries that can be added with a script tag found here.

<script src="https://ajax.googleapis.com/ajax/libs/jquery/1.11.3/jquery.min.js"></script>
<script src="my-app.js"></script>

For your JavaScript to use the jQuery object, it must come after jQuery is loaded into your document.

jQuery is an object with a bunch of methods associated with it that act as short cuts for commonly used JavaScript functionality. $ is also a reference to the jQuery object. Try typing both in the console. The basic syntax of a jQuery function looks like this:

$('selector').method()

Like in CSS, the selector refers to an HTML element. The action refers to the script that will be executed on that element.

jQuery's functionality basically overlaps with "vanilla" JavaScript. The main difference is that their methods can be shorter and simpler to write and can do more complex things with simple statements.

jQuery only uses methods (functions).

// jQuery
element.text("Hello");
element.append(childElement);

Vanilla JavaScript uses both methods and properties:

// vanilla
element.textContent = "Hello";  // property
element.appendChild(childElement);  // method

What's the deal with this syntax?

jQuery is used for a lot of events and the callback function structure looks similar to but is different from the .addEventListener we've been using. Let's compare.

var submit = document.getElementById("button");
submit.addEventListener("click", function() {
	console.log("Hello world!");
});
// submit.addEventListener("click", function);

In this syntax, submit is a reference to a HTML element. The method addEventListener takes to arguments, one is the specific event to look for, "click" and the seconds is the function to execute when the event occurs. The highlighted function is an anonymous function, it is itself an argument in the addEventListener function that is being declared at the same time.

In jQuery this syntax is similar but shorter:

$('#submit').click(function() {
	console.log("Hello world!");
});
$('#submit').click(function);

Because the anonymous function has an () operator and the {} code block, there are many nested parts, so it can be kind of confusing.
