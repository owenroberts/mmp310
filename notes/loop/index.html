A loop defines a set of commands to execute a certain number of times. JavaScript uses several loops statements, including do, while and for.

Loops look like other control statements, such as if () { ... } but run the code block a certain number of times.

while loop

A while statements executes as long as the first statement is true.

var i = 0;
if (i < 5) {
  i++;
  console.log(i);
}
var i = 0;
while (i < 5) {
  i++;
  console.log(i);
}

for loop

for (let i = 0; i < 5; i++) {
	console.log("This is the " + i + " loop.");
}

The for keyword indicates the beginning of the for loop. Inside the parentheses there are three statements that control the duration of the loop.

The first statement establishes the initial condition: let i = 0;.

The second statement establishes a conditional: i < 5;. As long as this condition evaluates to true, the loop will continue.

The final statement is the incremental/decremental statement or iteration: i++;.

for (initialization; conditional; iteration;) {
	loop body;
}

This is essentially like copying and pasting the same piece of code five times, while changing a number or input. Loops are an efficient way to execute actions on a list or array of data.

In many cases the initial condition will be the value 0 as in let i = 0; (i in this case is shorthand for index).

But our initial condition could be anything.

for (let i = 1; i < 6; i++) {
  console.log("This is the " + i + " loop.");
}
for (let i = 1, len = i + 5; i < len; i++) {
  console.log(i);
}
for (let i = 1, len = i + 5; i < len; i++) {
  console.log(i);
}

Likewise, we can change the conditional and iteration statements.

for (let i = 0; i < 100; i += 10) {
  console.log(i);
}
for (let i = 10; i > 0; i--) {
  console.log(i);
}

Example

A practical example of a loop in JavaScript:

var imgs = ["cat.jpg", "dog.jpg", "bird.jpg"];
for (let i = 0; i < imgs.length; i++) { 
	let img = new Image();
	img.src = imgs[i];
	document.body.appendChild(img);
}

Scope

Note on scope: in a for loop we use the let declaration to initialize i in order to scope i to the for loop.

for (let i = 0; i < 5; i++) {
	console.log(i);
}
console.log(i);

Look at difference between let and var.

for (var i = 0; i < 5; i++) {
	var x = i;
}
console.log(x);
for (var i = 0; i < 5; i++) {
	let z = i;
}
console.log(z);
