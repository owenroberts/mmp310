Today we're going to review some basic concepts introduced last semester.

We'll start with some JavaScript basics, data types, and variables.

Data types

The three JavaScript primitive data types we will use are: numbers, strings and booleans.

undefined and null are also primitive data types which represent variables that either do not have values or do not exist.

Integers and Floats

The numbers we have been using so far are all whole numbers, or integers, numbers that do not have a decimal point value.

JavaScript also uses floats, or numbers with decimal points.

3 / 4
// 0.75

However, JavaScript considers both integers and floats to be numbers. They are not separate types.

typeof 1
// "number"
typeof 0.5
// "number"

Strings

In JavaScript, single or double quotation marks are used to contain a string.

"I am a string."
'me too.'
typeof "I'm a string"

Booleans

Something that we often need to do in a program is make a decision based on different values. To do this, we can use boolean expressions, which evaluate whether a statement is true or false.

The == operator is used to evaluate a boolean statement.

1 == 1
// true
2 + 2 == 4
// true
2 + 2 == 5
// false

We can also compare strings and other values.

"hello" == "world"
// false
typeof 5 == 'number';
// true
var x;
x == undefined;
// true

Variables

Variables are short hand or key words that contains data. Variables in JavaScript can be used to contain any type of data. They can also be used to contain different types of data within one program.

JavaScript understands a few different types of text.

keywords words like function and var that are defined by the language.
variable names words for functions and variables that we create in our code like faceSize and faceColor.
hard data numbers and words that represent values like 0 or a string like "hello world".
operators symbols like = and + perform operations like assignment and arithmetic.

Creating a variable

The following is a simple statement that declares and initializes a variable.

var sentence = "Hello world";

Each part of our statement should be separated by one space.

Assignment operator

The = sign is for assigning values.

Unlike in math, it does not mean that two values are the same. It simply asks the program to store a value somewhere in the memory the computer and give that piece of memory a nickname.

When I say "gets" I mean the = assignment operator.

So this is what that statement would sound like:

A variable called sentence gets the value "Hello world."

Declaration and Assignment

Creating a variable requires two steps: declaration and assignment.

// declaration
var a;
// undefined

// assignment
a = 10;
// 10

A variables can be initialized by declaring and assigning in one statement:

// initialize
 var a = 10;

There are many reasons to use variables in code.

Variables make code more reusable.

Variables can be updated to make a program change through animation or interaction.

Variables can be used to make complex math and other calculations eaiser.

The following example uses a variable to repeat text in several places.

var welcome = "Welcome to MMP 310."; function setup() { createCanvas(640, 360); background(51); textSize(40); fill('plum'); text(welcome, 10, 50); fill('gold'); text(welcome, 10, 100); fill('aqua'); text(welcome, 10, 150); fill('coral'); text(welcome, 10, 200); fill('springgreen'); text(welcome, 10, 250); fill('rosybrown'); text(welcome, 10, 300); }

The next example demonstrates using a variable to animate a shape.

var r = 0; function draw() { background(51); rectMode(CENTER); // r is updated every frame r += PI * 0.01; // rotate by r around the center translate(width/2, height/2); rotate(r); fill('plum'); noStroke(); rect(0, 0, 100, 100); ellipse(0, 100, 50); ellipse(100, 0, 50); ellipse(-100, 0, 50); }
