This week, to save patterns created with the grid code, we will use the save() function in p5, which saves an image from the canvas graphics.

function setup() { createCanvas(640, 360); background('plum'); ellipse(width/2, height/2, 200); } // save the canvas when mouse is clicked function mouseClicked() { // is mouse on canvas ? if (mouseX > 0 && mouseX < width && mouseY > 0 && mouseY < height) { save(); } }

An optional argument can be added to specify the file name and type.

function setup() { createCanvas(640, 360); background('plum'); for (let x = 0; x <= width; x += 100) { ellipse(x, random(height), random(200)); } } function mouseClicked() { if (mouseX > 0 && mouseX < width && mouseY > 0 && mouseY < height) { save('loop.png'); } }
